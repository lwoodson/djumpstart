Djumpstarting Development

Djumpstart is a django application base for a common application stack. To use it, you will need to do the following:

  • Modify the database settings in env_settings.py
  • Change ROOT_URLCONF in settings.py
  • Modify the fabfile to suit the production environment
  • Modify base.html in core/templates
  • Modify index.html in core/templates
Tools added at startup:
  • pep8
  • pylint
  • nose
  • nose-django
  • south
  • django-debug-toolbar
